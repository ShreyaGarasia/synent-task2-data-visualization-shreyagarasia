# Task2 - Data Visualization

## Objective
Visualize patterns in dataset

## Dataset Details

| Attribute | Value |
|------------|--------|
| Dataset | Iris Dataset |
| Domain | Classification |
| Records | 150 |
| Features | Sepal Length, Sepal Width, Petal Length, Petal Width, Species |

## Tasks:
- Create:
- Bar chart
- Histogram
- Scatter plot
- Compare features

## Output:
Clear and meaningful visualizations

## Results
- Species distribution visualized
- Feature relationships identified
- Dataset patterns clearly observed

## Key Insights
- Petal Length and Petal Width have a strong positive relationship.
- Petal features clearly distinguish species.
- Visualizations help identify trends and patterns effectively.
